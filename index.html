Conhecimentos

HTML

A base de todo desenvolvimento Web é um bom HTML. É mais do que aprender tags, é saber a importância de um código semântico e bem estruturado. Em cima disso, aplico o layout CSS.

CSS

O mundo evoluiu, novas exigências surgiram e a Web cresceu junto. Com isso mergulhai nas técnicas e tecnologias mais recentes, desde pré-processamento de CSS com SASS, ferramentas de debug e layouts avançados com flexbox de multidispositivo.

Ruby on Rails

A linguagem Ruby foi minha primeira linguagem de programação e estou nela até hoje. Gosto muito de trabalhar com Ruby e o framework Rails que torno o processo de criação de software prazeroso e bastante agil.

Javascript

Utilizo JavaScript para controlar o HTML e o CSS e para manipular comportamentos na página. Aqui nesta página foi utilizado nos menus. Todo esse comportamento quem vai executar é o JavaScript.

Bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, melhorando a experiência do usuário em um site amigável e responsivo.

Git e GitHub

Git e Github são utilizados no meu dia a dia de desenvolvimentoe me ajudam na criação de software por um motivo bem simples: ter uma forma fácil de gerenciar o código fonte da aplicação, do sistema, do produto.
